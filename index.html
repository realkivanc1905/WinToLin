// Usage

Two commands.
That's all it takes.

01

Send a File

Run wintolin send <file> from any machine. WinToLin generates a receive command for the other side.

02

Receive a File

Run wintolin receive <ip:port> on the destination machine and the transfer begins instantly.

wintolin — demo session
# — Sender (Windows) —————————————————— PS C:\Users\User> wintolin send my_project.zip Preparing transfer... ✓ Listening on port 54321 ➤ Share with receiver: wintolin receive 192.168.1.10:54321 Waiting for connection... ✓ Connected! // Troubleshooting

Common Issues

⚠ Connection Refused

Make sure the sender is still running and waiting for a receiver. The sender must remain active until the receiver connects.

🌐 Transferring Over Internet

Open the displayed port on your router's Port Forwarding settings. Then share your Public IP (shown by WinToLin) with the receiver.
